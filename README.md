# Quantum
<p align="center"><img src="https://github.com/hex3gc/Quantum/blob/main/Images/expansionQt.png?raw=true" width=256></p>
<p align="center">Adds 3 new vanilla style equipment.</p>

<hr>

### Equipment
| Icon  | Description |
| ------------- | ------------- |
| <img src="https://github.com/hex3gc/Quantum/blob/main/Images/droneScrapEquip.png?raw=true" width=128> | **Build-It Kit**<br>```Convert items into parts to build new drone allies.```<hr><p align="right">Aim at a common, uncommon or legendary item and activate to convert it into **drone parts**. When you collect enough drone parts of a certain tier **[2/2/1]**, an **allied drone** of that tier will be added to your team. ***45s cooldown***</p> |
| <img src="https://github.com/hex3gc/Quantum/blob/main/Images/nullifier.png?raw=true" width=128> | **Nullifier**<br>```Corrupt an item into its void equivalent.```<hr><p align="right">Aim at an item with a **void equivalent** and activate to **corrupt** it into its void version. ***60s cooldown***</p> |
| <img src="https://github.com/hex3gc/Quantum/blob/main/Images/microwave.png?raw=true" width=128> | **Quantum Microwave**<br>```Electrocute nearby enemies to restore shields.```<hr><p align="right">Send 5 sets of **electrical shocks** for **3x200% damage** to random enemies within **35m**, **regenerating 30 shield** for each enemy zapped. **Grants 50% bonus movement speed while active.** ***45s cooldown***</p> |

### Recommended mods
- [Nautilus](https://thunderstore.io/package/hex3/Nautilus/) for a new set of void items to use with the Nullifier.
